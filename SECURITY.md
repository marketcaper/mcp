# Security Policy

Please do not open public issues for security reports.

To report a vulnerability or abuse risk related to Marketcaper MCP, email:

```text
info@marketcaper.com
```

Include:

- A short description of the issue.
- The affected public endpoint or documentation page.
- Steps to reproduce, if safe to share.
- Any suggested mitigation.

Do not include private keys, tokens, passwords, customer data, or unnecessary personal data in reports.

## Scope

In scope:

- Public MCP endpoint behavior.
- Public hosted MCP image endpoints.
- Public MCP docs and examples in this repository.
- Authentication bypass or data exposure affecting Marketcaper public services.

Out of scope:

- Trading advice or investment outcome claims.
- Bulk scraping attempts.
- Denial-of-service testing without prior approval.
- Reports that require accessing private infrastructure or non-public systems.

## Safe Use

Marketcaper MCP is intended for snapshots, comparisons, heatmaps, citations, and content workflows. It is not intended for trading automation, portfolio monitoring, backtesting, or bulk data extraction.
